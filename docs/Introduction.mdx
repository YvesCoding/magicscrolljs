**magic-scroll** is a scrollbar plugin based on React. You can customize scrollbar's style with this.
It can replace the native scrollbar for your react app.

## Features

- Basicly customize scrollbars, including rail, bar, scroll button.
- Typescript supported.
- SSR supported.
