# Get Started

## Installation

### Download from NPM

```bash
npm install magic-scroll -S
```

### Download from Yarn

```bash
yarn add magic-scroll
```

## Usage

**import magic-scroll**

```javascript
import Scrollbar from "magic-scroll"
```

**And then wrap the content by magic-scroll**

```javascript
<div class="parent-dom">
  <Scrollbar keepBarShow>
    <div class="child-dom" />
  </Scrollbar>
</div>
```

**And then, you will see a custom scrollbar like this.**

<Base keepBarShow />
