# Get Started

## Installation

### Download from NPM

```bash
npm install magic-scroll -S
```

### Download from Yarn

```bash
yarn add magic-scroll
```

## Usage

1. **import magic-scroll**

```javascript
import Scrollbar from "magic-scroll"
```

2. **And then wrap the content by magic-scroll**

```javascript
<div class="parent-dom">
  <Scrollbar keepBarShow>
    <div class="child-dom" />
  </Scrollbar>
</div>
```

3.That's all ! You will see the customize scrollbar show up !
